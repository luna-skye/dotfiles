## Options
Options can be accessed through the `zen.session.niri` config path.
| Name         | Type     | Default | Scope   | Description                               |
|--------------|----------|---------|---------|-------------------------------------------|
| `enable` | `boolean` | `false`   | NixOS Host | Whether to install the Niri window manager |
| `screenshotPath` | `string` | `"~/Pictures/Screenshots/screenshot_%Y-%m-%d_%H-%M-%S.png"`   | HM User | Where to store screenshots on disk and with what format |
