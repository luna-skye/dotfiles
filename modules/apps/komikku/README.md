## Options
Options can be accessed from the `zen.apps.komikku` config path.

| Name     | Type      | Default | Scope      | Description                                       |
|----------|-----------|---------|------------|---------------------------------------------------|
| `enable` | `boolean` | `false` | NixOS Host | Whether to install the Komikku manga/comic reader |
